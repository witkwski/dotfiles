### Installation:

It backups your old dotfiles into `~/.old_dots_backup` folder

```sh
$ git clone --recursive https://github.com/witkwski/dotfiles.git
$ cd dotfiles
$ ./setup

```

### Customs:

for customizing your env, add this:

```sh
touch ~/.dotfiles_custom
```
