# dotfiles

my minimalistic dotfiles.


## includes:
- zsh
- zsh-antigen
- zsh-syntax-highlighting
- zsh-autosuggestions
- tmux


## installation

it backups your old dotfiles into `~/.old_dots_backup` folder

```sh
$ git clone https://github.com/witkwski/dotfiles.git
$ cd dotfiles
$ ./setup.sh
```
